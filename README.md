# CodexFall

An informative wiki for exploring items, recipes, and other game content from Quinfall. This project provides a clean and efficient search interface to help players find detailed information about game items.

## Live Demo

Visit the live site: [https://maayk.github.io/CodexFall](https://maayk.github.io/CodexFall)

## Features

- **Item Search**: Quickly find items by name, type, or category
- **Recipe Database**: Browse and search crafting recipes
- **Detailed Item Information**: View comprehensive details about each item including stats, requirements, and usage
- **Multi-language Support**: Available in Portuguese (pt-BR), English (en-US), and Spanish (es-ES)
- **Dark Mode**: Built-in dark mode for comfortable viewing
- **Responsive Design**: Optimized for desktop and mobile devices
- **Privacy Focused**: No personal data collection from users

## Technologies

This site is built using modern web technologies:

- **Framework**: Vue 3
- **Styling**: Tailwind CSS
- **State Management**: Pinia
- **Internationalization**: Vue I18n

## How to Use

The Quinfall Wiki is designed to be intuitive and easy to use:

1. **Search**: Use the search bar to find items by name or keyword
2. **Browse**: Navigate through item categories using the sidebar or menu
3. **View Details**: Click on any item to see its complete information
4. **Filter**: Apply filters to narrow down your search results
5. **Language**: Switch between supported languages using the language selector

## Privacy

This application does not collect, store, or transmit any personal data from users. All data displayed is sourced from the game's public information and is used solely for informational purposes. The site runs entirely in your browser without tracking or analytics.
